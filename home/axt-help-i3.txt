i3 keyboard shortcuts

<W>+X                 -  Close Window
<W>+<enter>           -  Maximize Window
<W>+<cursor>          -  Move focus
<W>+>shift>+<cursor>  -  Move Window
<W>+<shift>+c         -  Reload i3
<W>+<shift>+r         -  Restart i3
<W>+<shift>+e         -  Exit i3
<W>+r                 -  Resize mode, resize with <cursor>, exit with <ESC>
<W>+t                 -  Start terminal
<W>+<space>           -  dmenu
